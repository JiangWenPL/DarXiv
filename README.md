# DarXiv
Decentralized arXiv.

##NOTE: 
All source code in this directory is under UTF-8 encoding and with LineFeed LF

##Usage:
After finishing configuration illustrated on the report.
You can run the program by following command in this directory.
```bash
truffle migrate
npm run dev
```
